## Personal Message Managment

##### Version
0.0.1

### Installation

First install node_modules for back-end:

```sh
$ npm install
```
Then install it for front-end
```sh
$ cd public
$ npm install
```

### Run

First Tab:
```sh
$ node start
```

To run webpack Tab:
```sh
$ npm install webpack -g
$ cd public
$ webpack
```
